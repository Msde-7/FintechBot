# FintechBot

A JavaScript and SQL-based Discord bot developed for Fintech at IU to provide real-time fund insights and promote engagement. The bot integrates with trading REST APIs and GroupMe to deliver automated daily updates and stock data on demand.

## Features
- Display live fund and individual stock performance
- Scheduled daily messages in Discord and GroupMe
- REST API integration for dynamic financial data
- SQLite backend for persistent data storage

## Tech Stack
- JavaScript (Node.js)
- SQLite
- Discord.js
- GroupMe API
- Custom REST API integrations
